Abra a consola e veja o resultado



vamos criar a seguinte string e manipulá-la: 
 
var str="Estudando Javascript";


 

toLowerCase(): converte toda a string para minúscula

 
console.log(str.toLowerCase());

 

toUpperCase(): converte toda a string para maiúscula

 
console.log(str.toUpperCase())

 

charAt(): recebe uma posição e retorna seu caractere dentro da string

 
console.log(str.charAt(10))

 

search(): recebe um caractere e retorna sua posição dentro da string

 
console.log(str.search('J'))

 

replace(): faz uma substituição de caracteres ,recebe aquele á ser substituido e depois o substituto

 
console.log(str.replace('Javascript','C#'))

 

slice(): faz um corte na string, recebe uma posição inicial do corte e a final e retorna o corte

 
console.log(str.slice(10,20));

 

substr(): também faz um corte na string, recebe uma posição inicial do corte e o tamanho do corte

 
console.log(str.substr(10,10));

 

split(): faz um corte na string em um ponto especifico (no caso abaixo ,no espaço) e adiciona cada um deles em um indice de um array

 
console.log(str.split(' '));

 

startsWith(): recebe um caractere e retorna true caso exista no inicio da string e false caso não exista no inicio


 
console.log(str.startsWith("E"));

 

endsWith(): recebe um caractere e retorna true caso exista no final da string e false caso não exista no final

 
console.log(str.endsWith("t"));

 

includes(): recebe um caractere e retorna true caso exista na string e false caso não exista

 
console.log(str.includes("e"));

 

 repeat(): recebe um número de vezes para repetir a string

 
var str="Estudando Javascript";
console.log(str.repeat(3));

 


length: retorna o tamanho da string,ou seja,o número de caracteres

 
console.log(str.length)
Podemos também juntar duas ou mais string que estão ou não guardadas em variáveis diferentes, vamos criar mais uma variável: 
 
var str2=" e node.js"
Agora vamos juntar as duas strings: 
 
concat(): junta duas strings estejam ou não dentro de variáveis
console.log(str.concat(str2));
console.log("estudando java".concat(" e spring Boot "));
